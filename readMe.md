## 🧠 What are HiCoPro and PrefDial?

- **🔍 HiCoPro** is a **Graph-Aware Framework** for **Hierarchical Dialogue Semantic Path Prediction**, enabling structured and context-aware understanding of multi-level dialogue semantics.
- **📚 PrefDial** is a large-scale, general-domain **Chinese hierarchical dialogue dataset** featuring systematic three-level annotations across **scene**, **task**, and **style**. It serves as a new benchmark for hierarchical dialogue understanding and preference-aware modeling.
